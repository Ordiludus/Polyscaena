# Polyscaena

Polyscaena (many stages/levels) is a collection of simple Java video game development software products including a general application programming framework, a custom image and audio format processing library and converter tool, and level creator and player utilities.

This is a hobby project and contributions will be made to it periodically under no fixed schedule. Any feedback or suggestions for inclusions in the project should be directed to [info@ordiludus.com](mailto:info@ordiludus.com). See the home page, [ordiludus.com](https://www.ordiludus.com), for additional project details and blog updates on development.
